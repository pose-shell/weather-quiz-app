# 概要

## 目的
気象予報士試験の学科試験を対象とした知識解説・問題演習サイトを作成する。
気象予報士試験対策の知識がためとして利用できるストレスの少ないサイトを作る

## 初期スコープ
- 学科専門のみ
- 四択の演習問題の作成
- 知識記事の作成
- 問題演習と知識記事を相互に行き来できるようにする
- ログインなし
- 学習履歴保存なし
- 問題・解説は自作
- 静的サイトとして公開
- 初期版では気象観測/数値予報のコンテンツを作成しのちに拡充していく
- 初期版では必修問題のみ取り扱い後々、細かい知識に対応した問題も拡充していく

## 初期スコープ外
- 学科一般
- 実技
- 会員登録
- コメント機能
- コミュニティ機能
